# Horiseon Refactor Project

A refactored version of the Horiseon marketing website focused on improving **accessibility**, **semantic HTML structure**, and **overall code quality**.  
This project ensures the site meets modern web standards and provides an inclusive experience for all users.

---

## 📌 Project Goals
- Improve accessibility for users and assistive technologies  
- Replace non-semantic HTML with semantic elements  
- Ensure logical structure independent of CSS  
- Add descriptive and meaningful `alt` attributes to images  
- Maintain sequential, accessible heading order  
- Provide a concise and descriptive `<title>` element  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  

---

## ✨ Improvements Made
- Converted `<div>` elements into semantic tags (`header`, `nav`, `main`, `section`, `footer`)  
- Added accessible `alt` text to all images  
- Ensured headings follow proper order (`h1` → `h2` → `h3`)  
- Cleaned up CSS selectors and consolidated duplicate rules  
- Improved overall readability and maintainability of the code
